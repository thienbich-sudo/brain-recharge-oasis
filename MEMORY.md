# Oasis Project - Agent Memory & User Preferences

## User Preferences
1. **Auto-Deploy Protocol (Active):** Mỗi khi hoàn thành xong tính năng hoặc chốt thay đổi (sau khi User đồng ý kết quả), *TỰ ĐỘNG* đóng gói, commit (`git add . && git commit`), và đẩy mã nguồn (`git push origin main`) lên GitHub để Vercel deploy. Tuyệt đối không đợi User nhắc nhở "Lưu dữ liệu đi" hay "Deploy đi". 
2. **Ngôn ngữ ưu tiên:** Tiếng Việt (chuyên nghiệp, mượt mà, hạn chế dùng từ triết lý khô khan, đặc biệt khi liên quan đến UX thư giãn/chữa lành).
3. **Môi trường:** Chạy test trên Cổng 5176 hoặc Vercel, luôn lưu ý vấn đề PWA Cache (hướng dẫn mở Ẩn danh nếu cache bị kẹt).
